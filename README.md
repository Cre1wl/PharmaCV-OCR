# 💊 PharmaCV-OCR

> **Система компьютерного зрения для автоматической проверки упаковок лекарственных средств**

Система предназначена для автоматизации контроля упаковки лекарственных средств в аптеках, логистических центрах и на фармацевтических складах. Позволяет быстро определить подлинность препарата и проверить срок его годности.

## 🎯 Возможности

| Функция | Описание |
|---------|----------|
| 📸 **Распознавание полей** | КП (GTIN), СН (S/N), Серия, Годен до |
| 🔍 **Проверка срока годности** | Автоматическое определение просроченных препаратов |
| 📊 **Визуализация** | Подсветка распознанных областей на изображении |
| ⚡ **Высокая скорость** | Обработка менее 2 секунд на изображение |
| 🌍 **Мультиязычность** | Поддержка русского и английского языков |
| 🖥️ **Веб-интерфейс** | Простой и интуитивный интерфейс на Gradio |
