# 💊 PharmaCV-OCR

Система компьютерного зрения для автоматического распознавания и проверки упаковок лекарственных средств.

## Краткое описание

Проект направлен на автоматизацию контроля качества фармацевтической продукции. Система позволяет быстро распознавать ключевые данные с упаковки лекарств: код продукции (КП/GTIN), серийный номер (СН/S/N), номер серии и срок годности. На основе распознанного срока годности программа делает вывод о том, годен препарат или просрочен.

## Возможности

- Распознавание текстовых полей с упаковки лекарств
- Детекция и выделение областей с ключевой информацией
- Проверка срока годности относительно текущей даты
- Поддержка русского и английского языков
- Визуализация результатов с подсветкой найденных полей

## Технологии

- YOLOv8 – детекция объектов
- EasyOCR – оптическое распознавание символов
- Gradio – веб-интерфейс
- OpenCV – обработка изображений
- Python 3.10+
